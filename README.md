**A fast, offline desktop clipboard manager for text templates.**

Tired of typing the same commands, messages, or code snippets over and over again?  
**MineCopier** lets you save them as tabs, organize them into groups, and copy with a single click.

Originally built **for Minecraft Map Makers**, but perfect for anyone who works with repetitive text — developers, moderators, support agents, content creators, and more.

---

## ✨ Features

### 📋 Tab Management
- **Create** tabs with a title, custom icon, and text
- **Edit** any tab — change the title, icon, or content anytime
- **Duplicate** tabs to quickly reuse templates
- **Delete** tabs with automatic group cleanup
- **One-click copy** — click the button and the text is in your clipboard
- **Custom icons** — PNG, JPG, GIF, BMP, TIFF supported, with auto-rounded corners
- **Default icon** — every tab gets a star icon if no custom one is set
- **Text preview** — see the content directly on the tab card

### 📁 Groups
- **Create groups** to organize tabs by category
- **Rename groups** whenever needed
- **Duplicate groups** with all tabs inside
- **Delete groups** — removes the group and all its tabs
- **"All" group** — view every tab across all groups in one place
- **Import/Export** — import and export groups via JSON

### 🔍 Search
- **Instant search** — type and results appear immediately
- **Match highlighting** — found words glow
- **Result counter** — know exactly how many tabs matched
- **Quick Search (Ctrl+Shift+C)** — a floating window on top of all apps
- **Keyboard navigation** — arrow keys to browse, Enter to copy
- **Paste from clipboard** — paste anything directly into the search bar
- **Clear button** — reset search with one click

### 🎨 Interface
- **Dark and Light themes** — switch between them anytime
- **Custom accent color** — RGB sliders with live preview, make it yours
- **5 languages** — 🇬🇧 English, 🇷🇺 Русский, 🇩🇪 Deutsch, 🇫🇷 Français, 🇪🇸 Español
- **Theme-adaptive icons** — icons switch between light and dark variants automatically
- **Context menus** — right-click actions for tabs and groups
- **Notifications** — subtle popups for copy, save, and other actions
- **Splash screen** — clean loading screen on startup
